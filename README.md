## Digital Platform for Software Development

### Case Study: Using SourceTree with GitHub

**Name:** Pakorn  
**Course:** CSI204 Digital Platform for Software Development

---

## Objectives

Practice using Git, GitHub, and SourceTree for version control.

---

## Steps Performed

### Step 1 : Create Git Repository
- Created a repository on GitHub.

### Step 2 : Clone Repository
- Cloned the repository to the local computer using SourceTree.

### Step 3 : Add Project Files
- Added `hello.txt` to the repository.

### Step 4 : Commit Changes
- Committed the project with the message:
  - `Add hello.txt`

### Step 5 : Push Changes
- Pushed the local repository to GitHub successfully.

### Step 6 : Pull Changes
- Pulled the latest changes from GitHub.

### Step 7 : Create Branch
- Created a new branch named `feature-test`.

### Step 8 : Update File
- Edited `hello.txt`
- Added the text:
  ```
  This is feature branch.
  ```

### Step 9 : Commit in Feature Branch
- Commit message:
  - `Update hello.txt in feature branch`

### Step 10 : Merge Branch
- Merged `feature-test` into `main`.
