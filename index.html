About the workshop

Project Overview

This workshop focuses on building a solid foundation in modern software collaboration tools. Students learn how to use Git for version control, GitHub for remote repository hosting and collaboration, and SourceTree as a visual Git client to simplify everyday version control operations.

In addition to source control, the workshop introduces Markdown for writing clean, structured documentation, and GitHub Pages for publishing static websites directly from a repository — exactly like the page you are viewing right now.

Why this matters

Analysis

⚠️

Problem

  • Manual file versioning leads to confusion and lost work.
  • Collaborating on code without a shared system is error-prone.
  • No standard way to publish or document a project online.
💡

Solution

  • Adopt Git for reliable, traceable version control.
  • Use GitHub as a central remote repository for collaboration.
  • Document with Markdown and deploy instantly via GitHub Pages.
✅

Benefits

  • Safer, organized, and fully trackable project history.
  • Smoother teamwork through branches and pull requests.
  • Free, fast publishing of professional documentation sites.
